This project presents a **basic exploratory dashboard** built using the Amazon Movies dataset.  
The goal of the dashboard is to analyze movie titles available on Amazon and uncover patterns related to **genres, release years, ratings, and content distribution**.
This project was created to practice **data analysis and visualization fundamentals** and to demonstrate how insights can be communicated clearly through dashboards.
**Dashboard Highlights**
The Dashboard provides highlights like:
-Total number of movie titles available
- Distribution of movies by **genre**
- Trend of movie releases over the years
- Content classification based on ratings
**Tools and Technologies used**
 Power BI
 Data exploration
 Data visualization
**Dataset**
 **Source**- Amazon Prime Titles(Kaggle)
**Key learnings**
 Understanding and preparing real-world datasets
- Creating meaningful visuals for storytelling
- Designing a clean and readable dashboard layout
- Extracting insights from categorical and time-based data.
